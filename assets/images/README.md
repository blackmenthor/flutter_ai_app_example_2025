# Required Haircut Images

You need to add the following image files to complete the preference screen:

## Hair Length Images (assets/images/hair_lengths/)

1. **very_short.jpg** - Buzz cut style (very short all over)
2. **short.jpg** - Classic short haircut 
3. **medium.jpg** - Medium length hair (versatile styling)
4. **long.jpg** - Long hair (shoulder length)
5. **very_long.jpg** - Very long hair (past shoulders)

## Hair Style Images (assets/images/hair_styles/)

1. **fade.jpg** - Fade haircut (gradual length transition)
2. **undercut.jpg** - Undercut style (short sides, longer top)
3. **buzz_cut.jpg** - Buzz cut (very short all over)
4. **crew_cut.jpg** - Crew cut (short, neat military style)
5. **pompadour.jpg** - Pompadour (swept back with volume)
6. **quiff.jpg** - Quiff (upward styled front section)
7. **slicked_back.jpg** - Slicked back (smooth, combed back)
8. **textured.jpg** - Textured cut (layered with movement)
9. **curly_wavy.jpg** - Curly/wavy natural hair
10. **natural_messy.jpg** - Natural messy/tousled style

## Image Requirements

- **Format**: JPG or PNG
- **Size**: Recommended 300x300px minimum
- **Aspect Ratio**: Square (1:1) preferred
- **Quality**: High resolution for crisp display
- **Copyright**: Use royalty-free images from:
  - Unsplash.com
  - Pexels.com
  - Freepik.com (with attribution if required)

## Usage in App

These images will be displayed in a grid layout where users can select their preferred hair length and style types. The images provide visual examples to help users understand each option before making their choice.
